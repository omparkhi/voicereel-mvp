# VoiceReel India 🎬

> India's first AI-powered Hindi Reel maker 
> for small businesses and regional creators.

## What We're Building
Convert any topic into a professional Hindi/Marathi/Tamil 
Reel in under 2 minutes — automatically.

No camera. No editor. Just your topic.

## Core Stack
- Frontend: Next.js 14 + Tailwind CSS
- Backend: Node.js + Express + MongoDB
- Voice: Sarvam Bulbul v3 TTS
- Translation: Sarvam Translate API
- Script: Claude API (Anthropic)
- Avatar: D-ID API
- Storage: Cloudflare R2
- Payments: Razorpay

## Target Market
- 63M Indian SMBs (Salons, Clinics, Restaurants)
- Regional Content Creators (Hindi/Marathi/Tamil)
